# Assembly Programming Fundamentals

A collection of x86 Assembly Language programs developed as part of laboratory assignments and coursework. This repository demonstrates fundamental assembly programming concepts including indexed addressing, array manipulation, loops, procedures, register usage, and user input/output operations.

## Overview

This project contains solutions to several assembly programming exercises designed to strengthen understanding of low-level programming and computer architecture concepts.

## Implemented Programs

### 1. Array Gap Sum Calculator

Calculates the total gap between consecutive elements in a non-decreasing word array.

**Example Array:**

```
{0, 2, 5, 9, 10}
```

**Gap Calculation:**

```
2, 3, 4, 1
```

**Total Gap:**

```
10
```

### 2. Number Sequence Generator

Generates a decreasing sequence pattern based on an initialized value.

**Example Output:**

```
1 2 3 4 5 6 7 8
2 3 4 5 6 7 8
3 4 5 6 7 8
4 5 6 7 8
5 6 7 8
6 7 8
7 8
8
```

### 3. Array Sum Program

Prompts the user to enter three 32-bit integers, stores them in an array, computes the total sum, and displays the result.

### 4. Grade Letter Procedure

Receives an integer between 0 and 100 and returns the corresponding letter grade.

| Score Range | Grade |
| ----------- | ----- |
| 90 – 100    | A     |
| 80 – 89     | B     |
| 70 – 79     | C     |
| 60 – 69     | D     |
| 0 – 59      | F     |
